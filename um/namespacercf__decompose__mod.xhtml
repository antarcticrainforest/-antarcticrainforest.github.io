Subroutine Rcf_Decompose - performs grid decomposition.

This routine performs a 2D decomposition - taking the global X and Y data sizes and decomposing across nproc_EW processors in the X direction and nproc_NS processors in the Y direction. Method: Based on UM4.5 code but now uses F90 data-structures. This file belongs in section: Reconfiguration
