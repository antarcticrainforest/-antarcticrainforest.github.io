Subroutine Rcf_Write_Multi - parallel interface to buffout.

This routine provides an interface to BUFFOUT for the parallel Reconfiguration. It is used where each process must write out a local section of a global field. Method: Each processor sends its local part of the global field to PE 0 which assembles all the parts, and then writes them to disk. Fields which are compressed to land points are expanded before sending to PE 0, PE 0 then compresses the global field before writing it to disk.
