Initialise aerosol index

Select Aerosol/Cloud types to be used - define types here

MIEDX(3) = 20 !> Dust - large particles MIEDX(4) = 17 !> Dust - sub-micron MIEDX(5) = 25 !> Sulphate MIEDX(6) = 22 !> Black carbon MIEDX(7) = 23 !> Organic Carbon MIEDX(8) = 24 !> Sea-Salt

sulphur is used as 3. aerosol. Additional aersol types are added in set_prof, please add here the corresponding MIEDX values and change MX in mo_fastj_data.f

Convert from 550nm to 1000nm optical depths (dependent on aerosol type) Necessary for aerosol climatology only
