Subroutine SOLPOS !> Code Owner: See Unified Model Code Owners HTML page This file belongs in section: Radiation Control Purpose : Calculations of the earth's orbit described in the first page of the "Calculation of incoming insolation" section of UMDP 23, i.e. from the day of the year (and, in forecast mode, whether it is a leap year) and the orbital "constants" (which vary over "Milankovitch" timescales) it calculates the sin of the solar declination and the inverse-square scaling factor for the solar "constant". It is thus intrinsically scalar.
