Subroutine Rcf_Create_Dump - main dump creation loop.

Creates the output dump based on the input choices. Method: Data sources are setup and then fields in the output dump are looped over. Each one is set appropriately ( interpolated from input dump, set to 0/MDI/constant/from file/etc). Pre and Post processing is performed as required.
