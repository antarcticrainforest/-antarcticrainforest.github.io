subroutine rcf_readumhdr_mod::rcf_readumhdr ( type (um_header_type), intent(inout)  UMhdr)

Subroutine Rcf_ReadUMhdr - read a header from the dump.

Read in a model header from a UM dump. Method: Call READ_FLH (UM routine) to read in Fixed-length Header. Call Rcf_Allochdr to allocate array sizes for whole header, and Call READHEAD (UM routine) to read in whole header. Based on VAR code.
