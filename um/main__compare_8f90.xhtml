Program MAIN_COMPARE and Subroutine COMPARE Purpose: Compares two UM atmosphere, ocean, or ancillary files. MAIN_COMPARE reads in fixed length and integer headers of UM files to be compared, extracts dimensions of each file and then passes these values to subroutine COMPARE. COMPARE subroutine: Compares two UM atmosphere, ocean, or ancillary files. COMPARE reads in headers and data fields from files on NFTIN1 and NFTIN2, comparing values. UNIT 6: If an exact compare is found the message 'OK' is written out, otherwise i) if header, all differring values are printed ii) if field, 1st 10 differring values are printed plus the maximum difference between the fields. iii) if field only present in one file, a warning message is displayed UNIT 7: Number of differences displayed for each header.
