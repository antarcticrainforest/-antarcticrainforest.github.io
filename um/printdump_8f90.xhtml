SUBROUTINE PRINTDUMP Purpose: Prints a summary of contents of atmosphere, ocean or ancillary file. PRINTDUMP reads in headers and data fields from unit NFTIN printing out a summary of their contents. Printout of headers is written to unit 7. Printout of data fields is written to unit 6. !> Documentation: UM Doc Paper F5 System Tasks: F3,F4,F6 -!> Arguments:!> Code Owner: See Unified Model Code Owners HTML page This file belongs in section: Small execs
