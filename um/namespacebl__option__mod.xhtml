Detailed Description

*****************************COPYRIGHT******************************** *****************************COPYRIGHT******************************** *****************************COPYRIGHT********************************

Data module for switches/options concerned with the BL scheme. Module containing runtime options/data used by the boundary layer scheme Method: Switches and associated data values used by the boundary layer scheme are defined here and assigned default values. These may be overridden by namelist input. Any routine wishing to use these options may do so with the 'USE' statement. This file belongs in section: Boundary Layer Language: FORTRAN 90
