◆ l_var_entrain

logical cv_dependent_switch_mod::l_var_entrain = .false.

Module containing extra switches set dependent on convection namelist variables read in at the start of the model.

Method: The values of the variables are overriden by a call to the routine cv_set_dependent_switches called from readlsta or scm_shell after reading in namelists. Doing this should save CPU as the switches!> will not be set under if tests each call to e.g. layer_cn or glue_conv Language: Fortran 90. This code is written to UM programming standards version 8.1. Declarations: Giving default values of switches. Extra switches controlling entrainment and detrainment in layer_cn Used by 5A code only at present
