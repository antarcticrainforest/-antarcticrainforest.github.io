Subroutine Rcf_Set_Neighbour - What are my neighbouring processors?

This routine finds the tids of the North, South, East and West neighbouring processes. It takes account of the boundary condition in each dimension (X and Y) which can be either: cyclic : wrap around static : no wrap around Method: The tid of each neighbouring process is calculated (taking into account the relevant boundary conditions) and placed in the neighbour array. Derived from UM4.5 code. This file belongs in section: Reconfiguration
