If it is free then that unit is returned. ! If one cannot be found then ereport is called.
34  ! NOTE: If using this routine to define a file unit for use in either ! reading or writing files, you should be aware that the file unit ! returned may not be the same if another peice of code ! functionality is used that also calls this routine. You should ! therefore always refer to your open file with reference to the ! dynamically defined unit, rather than with the use of a ! hard-coded unit number.
