=============================================================================

Holds fixed dimensions for variables read in via SCM namelists. Fotran 90 does not allow namelist array members to have varying dimensions, so they must be dimensioned using fixed parameters. This file belongs in section: Single Column Model Language: Fortran 90. This code is written to UMDP3 v8.3 programming standards. =============================================================================
