logical cv_cntl_mod::lcv_3d_ccw = .FALSE.

Module containing logical switches from &CNTLATM which control the convection code.!> Method: Switches are declared initialised with the corresponding values set in CNTLATM. The mod cannot use the same names as in &CNTLATM because the include file cntlatm.h occurs in too many routines to make it practical.
