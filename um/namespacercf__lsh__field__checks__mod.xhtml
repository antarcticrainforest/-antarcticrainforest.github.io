Subroutine Rcf_Lsh_Field_Checks - Check LSH initialisation settings.

Make consistency checks on the options set for initialising the large-scale hydrology prognostics. Method: If the fields for mean and standard deviation of topographic index are not taken from the start dump or are interpolated from the start dump, those fields which are calculated directly from them much be reset. Check that variable fexp (stashcode 276) is set to be a global constant. Check that none of the LSH variables fexp are set to zero of missing data.
