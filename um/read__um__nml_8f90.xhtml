Routine to read in SIZES control file generated by the UMUI. The namelists NLSIZES and STSHCOMP contain lots of variables that are not applicable to the SCM. To use the same namelists the UM generates without keeping the extra fields, the NLSIZES and STSHCOMP nameslists are read in this routine then only the required fields are passed to scm_shell.
