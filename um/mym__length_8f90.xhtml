SUBROUTINE mym_length SUBROUTINE mym_length Purpose: To calculate mixing length in the MY model. The square root of TKE, required by this subroutine and elsewhere, is also returned.

Programming standard : unified model documentation paper No 3 Documentation: UMDP 024A This code is based on the code provided by the authors who wrote the following papers.

  • Nakanishi, M. and H. Niino, 2009: Development of an improved turbulence closure model for the atmospheric boundary layer. J. Meteor. Soc. Japan, 87, 895-912.
  • Nakanishi, M. and H. Niino, 2006: An improved Mellor-Yamada Level-3 model: Its numerical stability and application to a regional prediction of advection fog. Boundary-Layer Meteor., 119, 397-407.
  • Nakanishi, M. and H. Niino, 2004: An improved Mellor-Yamada Level-3 model with condensation physics: Its design and verification. Boundary-Layer Meteor., 112, 1-31.
  • Nakanishi, M., 2001: Improvement of the Mellor-Yamada turbulence closure model based on large-eddy simulation data. Boundary-Layer Meteor., 99, 349-378. The web site publicising their code: http://www.nda.ac.jp/~naka/MYNN/index.html This file belongs in section: Boundary Layer
