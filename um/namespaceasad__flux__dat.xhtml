Set to 0 otherwise. !> #SPECIES = number of species. IF TP==RXN this is total of reactants + products IF TP!>=RXN this should be 1 ONLY R1 = IF TP!>=RXN this is the species being deposited/emitted NOTE: If the same stash number is specified for multiple diagnostics then these will be summed up before outputting into stash for processing.
