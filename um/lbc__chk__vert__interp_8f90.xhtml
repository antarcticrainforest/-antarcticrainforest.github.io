subroutine lbc_chk_vert_interp (   include)

Check if Vertical Interpolation is required for LBC data.

Check if Vertical Interpolation is required for LBC data Subroutine Interface: This file belongs in section: MakeBC

Determine whether vertical interpolation of Lateral Boundary conditions is required between model and LBC levels.

Method: Vertical interpolation is switched on if any of the following are different.

  • No of model or wet levels.
  • First rho level at which height is constant.
  • Height of top of model.
  • Eta values for theta or rho levels.

Language: FORTRAN 77 + common extensions. This code is written to UMDP3 v6 programming standards.
