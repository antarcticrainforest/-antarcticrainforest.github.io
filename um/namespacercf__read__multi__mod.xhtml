Subroutine Rcf_Read_Multi - parallel interface to buffin.

This routine provides an interface to BUFFIN for the parallel reconfiguration. It is used where each process must read in a local section of a global field. Method: PE 0 reads in the global field, and then distributes the relevant parts of it to each processor. Fields compressed to land points are expanded by PE 0, and recompressed after being received by the relevant processor.
