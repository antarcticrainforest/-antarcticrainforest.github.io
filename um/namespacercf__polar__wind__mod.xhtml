Subroutine Rcf_Polar_Wind - Averaging of Polar wind rows.

Performs a vector mean on Polar wind rows. Method: Magnitude and direction of non-polar rows are calculated and used to set adjecent polar rows. Normally this is using v to estimate u at pole but ENDGAME requires v at poles so uses u to calculate v. This file belongs in section: Reconfiguration Language: FORTRAN 90 This code is written to UMDP3 v6 programming standards.

Local Variables.

Section 1. Calculate magnitude and direction of polar vector wind from non-polar component of wind on row around pole. !> Only need data at poles

Calculate start index in data depending on which pole we are. Pole_dir gives the sign of the result (cos(t)+sin(t)) after!> rotating a vector.
