subroutine scm_substep_start ( integer  substep_number)

Signal start of sub-step to SCM output diagnostic system.

Signal start of sub-step to SCM output diagnostic system

Parameters
substep_numberLanguage: Fortran90

Record the sub-step number in SCMop

Theoretically the number of sub-steps should be contained in SCMopnum_substeps, which is set in setup_diags. But in case there's some unconventional sub-stepping going on somewhere we'll adjust this value to take account of any higher-than-expected sub-step numbers.
