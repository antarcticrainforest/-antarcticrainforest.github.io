Subroutine Rcf_ReadLSMin - read land-sea masks.

This module is designed to read-in and setup an input land-sea mask for multiple PEs. IE, both local and global LSMs are read and appropriate sizes calculated. It is assumed that space has already been allocated for them, but this is checked. Method: Local LSM is read in on each PE. This is gathered on PE 0 and broadcast so that all pes have a copy of the global LSM. This file belongs in section: Reconfiguration
