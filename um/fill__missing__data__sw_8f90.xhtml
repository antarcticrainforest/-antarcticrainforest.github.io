Subroutine to fill in missing data.

Subroutine to fill in missing data Method: Radiative fluxes may not have been calculated at all points: we now fill in as required. This part of the code was originally located in RAD_CTL2 (v6.1 and below) but has been move into a subroutine in order to make RAD_CTL2 more readable. This file belongs in section: Radiation Control FORTRAN 77/90 with extensions listed in documentation.
