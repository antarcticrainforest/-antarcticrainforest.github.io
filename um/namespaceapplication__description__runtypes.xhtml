Detailed Description

Purpose: The application names and run-time types.

Purpose: The application names and run-time types. Used to enable the UM code to determine!> the executable target (program main), build type, sizes, and parallel nature. This facilitates reduction in preprocessor based compilation in favour of software switching Code Owner: See Unified Model Code Owners HTML page This file belongs in section: Top Level
