subroutine s_physwitch::read_physwitch ( )

=============================================================================

Allows SCM to read in PHYSWITCH namelist from forcing file, scm_nml. PHYSWITCH contains information on the different physics modes. Method: Namelist PHYSWITCH is defined in this module and read in by contained subroutine read_physwitch. Scalar variables are read directly to s_main_force. Array variables are declared as private single rank arrays!> of fixed length (arrays cannot be varying length in namelists) and reshaped before being transferred to arrays of the correct size/shape in s_main_force. This file belongs in section: Single Column Model Language: Fortran 90.
