Detailed Description

SUBROUTINES LS_PPN and LS_PPNC SUBROUTINES LS_PPN and LS_PPNC Purpose: LS_PPN and LS_PPNC: Calculate large-scale (dynamical) precipitation. LS_PPNC is the gather/scatter routine which then calls LSP_ICE. Note: in all cases, level counters (incl subscripts) run from 1 (lowest model layer) to qdimsk_end (topmost "wet" model layer)
