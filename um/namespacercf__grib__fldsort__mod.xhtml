SUBROUTINE rcf_Grib_FldSort Establish what information is contained within the GRIB file.

GRIB field headers in either ascending or descendng order of any one parameter in Block_0 Method: Quite simply, Bubblesort. It compares 2 adjacent entries and if they aren't in the correct order (relative to each other) swap all the pointers so they appear in the opposite order within the list. Repeat this process as you traverse through the list. Then go back to the list start and repeat again until no change occured during a complete pass through the list.
