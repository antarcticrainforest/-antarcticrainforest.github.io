Subroutine Rcf_Adjust_Pstar Adjusts P* when an ancillary orography has been used with an interpolated P*.

A similar method to that used in 4.5 is employed, with a change to a height based vertical co-ordinate frame. Method: Assuming a constant lapse rate, a surface temperature (Ts) is calculated based on a temperature at a reference height (Tr) Ts = Tr + lapse * (Zr - Z0_at_old_orography) P* is then adjusted thus:!> P*' = P* [ Ts - lapse ( Zr - Z0_new_orography) ] ** g/(R lapse) [ – ] [ Ts ] This file belongs in section: Reconfiguration Language: FORTRAN 90 This code is written to UMDP3 v6 programming standards.
