Subroutine eg_pofil_vatp.

Subroutine eg_pofil_vatp Purpose: Filter/diffusion based on first order conservative diffusion operator. Applied as multiple EW 1-2-1 filter near poles. Method: For any field. Pointers used to input grid This version tested for v-at-the-poles/ENDGAME This file belongs in section: DIFfusion and Filtering Language: FORTRAN 77 + CRAY extensions This code is written to UMDP3 programming standards.
