Detailed Description

Input control for radiation. Input control for radiation.

Module containing input switches/settings as used by the radiation code. This file belongs in section: Radiation Control Method: Switches are initialised to false and read in from the UMUI. The module may then be used directly where the switches are needed within the radiation code. Language: FORTRAN 90 This code is written to UMDP3
