PROGRAM MAIN_CONVIEEE!> Purpose: Converts a dump, ancillary or fieldsfile (atmosphere or ocean) between different into 32-bit or 64-bit IEEE format or vice-versa. The following conversions are supported:!> On a IEEE machine 64-bit IEEE to 32-bit IEEE 64-bit IEEE to 64-bit IEEE In either case, WGDOS data will be unpacked if requested. MAIN_CONVIEEE reads in fixed length and integer headers of UM file to be converted, extracts dimensions of file and then passes these values to subroutine CONVIEEE.
