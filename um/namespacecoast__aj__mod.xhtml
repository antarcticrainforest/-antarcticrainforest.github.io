Detailed Description

SUBROUTINE COAST_AJ-!> SUBROUTINE COAST_AJ-!>

Purpose: (i) To produce gather indices which map each coastal point on the target grid onto its nearest point on the source grid. This allows correction of those surface fields which are non-homogeneous across land/sea boundaries after horizontal interpolation by subroutine H_INT. The algorithm uses linear interpolation weights and gather indices calculated by subroutine H_INT_CO. (ii) If a land-sea mask for the target grid is not provided, one is created. When a target land/sea mask is provided, further index is output containing those points on the target grid for which the 4 surrounding source points are not of the same land/sea type as the target point. These points will generally be new islands etc resolved b a hires land/sea mask. They should be set to appropriate values (eg climatology) as required.
