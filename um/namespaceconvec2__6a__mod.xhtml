Completes lifting of the parcel from layer k to k+1.

Calls subroutines parcel and environ Subroutine parcel carries out the forced detrainment calculation, tests to see if convection is termintating!> and calculates the precipitation rate from layer k+1. Subroutine environ calculates the effect of convection upon the!> large-scale atmosphere. !> The parcel properties at level k+1 are then fully updated. CAPE and!> dCAPEbydt are calculated for use in the CAPE closure. Method: See UM Documentation paper No 27 This file belongs in section: Convection Language: Fortran 90. This code is written to UMDP3 v8.3 programming standards.
