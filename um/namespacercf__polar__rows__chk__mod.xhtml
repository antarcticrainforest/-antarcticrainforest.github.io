Subroutine Rcf_Polar_Rows_Chk.

This subroutine takes a global dump and checks the north and south polar rows of every level of every field on the P grid. If it finds that such a row contains non-uniform data it aborts the job, otherwise nothing happens. Method: Assume the rows are uniform and run through each field level by level, comparing entries against a base value and noting any inconsistencies.
