SUBROUTINE Run_Init Purpose: Called by scm_main (Single Column Model main routine) to do initialisations. Language - FORTRAN 90 This file belongs in section: Single Column Model =====================================================================

OPTIONS TO SET INITIAL PROFILES

(i) Observational large scale forcing (OBS=TRUE of namelist LOGIC) Initial data is then from namelist INPROF (ii) Statistical large scale forcing (STATS=TRUE of namelist LOGIC) Initial data can either be derived from climate datasets using subroutine INITSTAT or set from namelist INPROF (set ALTDAT=TRUE in namelist LOGIC) (iii) No large-scale forcing initial data is set fron namelist INPROF (iv) Continuation from previous run stored on tape (Set TAPEIN=TRUE in namelist LOGIC). All other initial data

is overwritten
