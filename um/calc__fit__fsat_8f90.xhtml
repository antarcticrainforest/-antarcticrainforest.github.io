Subroutine CALC_FIT_FSAT!> Subroutine CALC_FIT_FSAT!> Purpose: To speed up the large scale hydrology code (LTOP=TRUE) dramatically. This is done by fitting exponential functions to the incomplete gamma function for each grid box and the complete range of possible "water table" (top_crit) cases - see documentation. Estimates the fitted parameters for Fsat=function(ZW) and Fwet=function(ZW) for each land grid point. (Calculating the incomplete gamma function for each grid box at each time step was very time consuming). !> Documentation: UNIFIED MODEL DOCUMENTATION PAPER NO 25.
