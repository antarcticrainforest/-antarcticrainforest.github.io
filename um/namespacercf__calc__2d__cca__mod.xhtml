Subroutine Rcf_Calc_2D_CCA - calculates 2D cca from 3D cca.

convective cloud amount array. Method: If an anvil is detected (ie there is more convective cloud at the top than the base), the base value is divided by the tower_factor. Otherwise, the value is the one at the cloud base. Note that the tower_factor used to genenerate the 3D field is not available, so the most common one in practice is used.
