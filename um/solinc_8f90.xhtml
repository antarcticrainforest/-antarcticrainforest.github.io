Correction for the angle of solar incidence on sloping terrain.

Correction for the angle of solar incidence on sloping terrain. Calculate the orographic correction to be applied to the Direct SW flux at the surface using the angle of solar incidence on sloping terrain. Uses data from global data module 'solinc_data'. Method: Spherical trigonometry. This file belongs in section: Radiation Control Language: FORTRAN 90 This code is written to UMDP3 v7.4 programming standards.
