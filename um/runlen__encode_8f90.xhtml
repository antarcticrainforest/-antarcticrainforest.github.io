Purpose: Reduce storage requirements by removing land points from fields within ocean fieldsfiles. This is done using run length encoding to compress the sequences of missing data values that represent the land points. Using this method, a sequence of missing data values are mapped into a single missing data value followed by a value indicating the length of the run of missing data values.
