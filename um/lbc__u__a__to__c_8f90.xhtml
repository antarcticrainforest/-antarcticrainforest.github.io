Interpolate wind components from P to U/V grids.

Interpolate wind components from P to U/V grids. Interpolates u-component from the p-grid to u-grid. Method: u-comp and v-comp are both at all p-grid points after being rotated. The u_comp is interpolated in the x_direction to the u-grid. The v_comp is interpolated in the y_direction to the v-grid. Linear interpolation is used. The u-grid and v-grid are within the p-grid so there is no approximation round the edges. This file belongs in section: MakeBC Language: FORTRAN 77 + common extensions. This code is written to UMDP3 v6 programming standards.
