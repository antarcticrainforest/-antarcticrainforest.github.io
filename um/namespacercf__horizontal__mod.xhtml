Subroutine Rcf_horizontal - horizontal interpolation.

This module contains a wrapper subroutine for horizontal interpolation. The interpolation is done level-by-level but on a vertically decomposed grid. Thus we need to gather data by levels onto the compute PEs and then rescatter it onto the output grids. This won't be the fastest routine in the world, but should be reliable and be easily verifiable for bit-reproducibility etc. Method: Note that land compressed fields are uncompressed and recompressed, copying is done if that is all that is required, coastal adjustment is performed and polar row averaging is done.
