integer, parameter rcf_fortranio_mod::lb = 10

Subroutine Rcf_Get_Unit - obtains a unit number for IO Subroutine Rcf_Free_Unit - frees up a unit number again.

The module contains data and routines to manage the dynamic allocation and freeing of unit numbers for Fortran IO. Method: An array UnitsInUse holds a list of all unit numbers already allocated through the modules subroutines. This is based on code used by the VAR group.
