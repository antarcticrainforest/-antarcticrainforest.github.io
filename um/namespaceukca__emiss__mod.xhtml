Method: Declares the emissions data structure where the emission values and other relevant data are stored. Most of the!> emission fields are read from NetCDF files, but others are calculated online. 1. UKCA_EMISS_INIT: Identify fields in NetCDF emission files and ititialise emissions structure accordingly.

  1. UKCA_EM_STRUCT_INIT: Initialise emissions structure to!> default values.
  2. UKCA_EMISS_UPDATE: Check if time to update values in emissions structure and do it when needed.
  3. GET_EMFILE_REC: Get time records from a NetCDF file just before and after the model time.
