118 ! Initialise DAMP_FACTOR to 1.0
119 ! Note that for RESIST_B_TILE, negative values have already been
120 ! set to in SFEXCH8A, but we repeat it here for safety.
121 ! For STR_RESIST_S values depend on surface type (land, sea,
122 ! snow, ice) as well as tracer identity. First calculate stomatal
123 ! resistance (=1/conductance, avoiding dividing by zero).

132  END IF

135  ELSE

137  END IF

143 ! Two special cases need to be trapped here. The canopy capacity
144 ! (CATCH) is zero at land ice points, so exclude these from the
145 ! calculation. Also, there is a possibility that canopy water may
146 ! exceed canopy capacity due to leaves having fallen, so take care
147 ! of this too.

177 ! Need to set STR_RESIST_S to suitable values over snow and ice.
178 ! Where there is snow cover, calculate an approximate snow fraction
179 ! for the tile using the formula 1-exp(-ASNOW*SNODEP)
180 ! Note that for atmospheric model run there should not be any sea
181 ! points with SNODEP > 0, and land_ice points (including Antarctica)

193 ! Loop over all land tiles:
194 ! (Note: Routine TILEPTS sets array elements TILE_PTS(N) to the no. of
