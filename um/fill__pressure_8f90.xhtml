End of header -

Copy input field to dummy field ready for checking the field and making adjustments.

If input field has a missing value at point of interest but mask field indicates Cb, check if there is an input field value at the southerly, northery or easterly neighbouring gridpoints. If so, set value at point of interest to be this value. Repeat check through field a further 9 times.

Optimisation comments: kk Exchanging I and J loop avoids bank conflicts AND allows vectorization.
