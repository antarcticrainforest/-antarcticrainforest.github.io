This section relates to aspects of the netcdf file Included mainly for historical reasons. Data is now required to be in data format 0 Where is the data from (0=BDAN/Reading, 1 = BADC) This affects a couple of things i) Data starts at top of the atmosphere in BDAN data so have to flip the data to get in desired (BADC) format ii) surface pressure is 2d+1 rather than just 2D so have to use slightly different routines iii) naming conventions differ (see below) More...
