subroutine define_storm (gwp, storm_field, nspts)
 

Detailed Description

Purpose: Within the electric scheme, this routine defines whether.

Purpose: Within the electric scheme, this routine defines whether a storm exists within each model column based on a given definition of 'a storm'. It then sets a 2D logical (storm_field) with the .TRUE. condition being used to determine a storm point. The current definition for a storm is based entirely on graupel water path (gwp) but the code is flexible to add other definitions in future. This file belongs in section: Electric
