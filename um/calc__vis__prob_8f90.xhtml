Calculates visibility probability.

Calculates visibility probability. The visibility probability is similar to the cloud fraction except it records the fraction of a grid box with RH greater than that required for the critical visibility (e.g 1 km), taking into account precipitation. Suitable for single-column use. This file belongs in section: Atmosphere Service This code is written to UMDP3 standards. Subroutine Interface:
