Purpose: Calculate form drag profiles for distributed Purpose: Calculate form drag profiles for distributed drag parametrization.

Based on work by Wood, Brown and Hewer (2001), Quart. J. Roy. Met. Soc., 127, 759–777. Programming standard: UMDP 3 Turbulent form drag due to sub-grid orography Method: Calculates the form drag and stress profiles due to sub-grid scale orography. The stress is later added as an additional explicit stress to the boundary-layer equations in EXFXUV8A. The orographic stress is added to the surface stress in BDYLYR8A. This file belongs in section: Boundary Layer Language: FORTRAN 77 + common extensions. This code is written to UMDP3 v6 programming standards.
