Detailed Description

This file belongs in section: MPP.

This file belongs in section: MPP This module provides methods for updating boundaries (halos) on fields, either by extending the field or exchanging data with neighbouring processors depending on the model. Communications take place in private communicators which musch be!> initialised with a call to halo_exchange_init() before other methods can be used.!> Logic and communicator setup is non-trivial for global models.
