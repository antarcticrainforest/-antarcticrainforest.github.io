Routine: STZONM!> Purpose: Calculate weighted zonal mean within a region specified by a lower left hand and upper right hand corner. (STASH service routine). Programming standard: UM Doc Paper 3, version 2 (7/9/90) Logical components covered: D713 Project task: D7 External documentation: Unified Model Doc Paper C4 - Storage handling and diagnostic system (STASH) Interface and arguments:!> Code Owner: See Unified Model Code Owners HTML page This file belongs in section: STASH

pstar_weight and area_weight arrays contain appropriate weighting factors, interpolated to the correct grid, for mass weighting and area weighting respectively. If either type of weighting is not required, the relevant array is set to 1.0 The mask array contains appropriate masking
