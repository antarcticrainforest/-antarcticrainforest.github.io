Routine: STEXTC!> Purpose: Extracts a weighted subfield within a region specified by a lower left hand and upper right hand corner. Single level at a time. (STASH service routine). External documentation: Unified Model Doc Paper C4 - Storage handling and diagnostic system (STASH) Code Owner: See Unified Model Code Owners HTML page This file belongs in section: STASH

Local variables

Calculate the output field, by multiplying the input field by pstar_weight and area_weight. These arrays contain appropriate weighting factors, interpolated to the correct grid, for mass weighting and area weighting respectively. If either type of weighting is not required, the relevant array is set to 1.0
