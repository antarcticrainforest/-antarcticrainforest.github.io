Routine that computes the hydrometeor effective radius of the precipitating hydrometeors.

Method: Uses the analytical solution of the ith moment of the PSD to compute the ratio between the 3rd and 2nd moments, divided by two. This is explained in the COSP user's manual. It used the parameters that describe the PSD in the UM microphysical settings (UMDP26). This file belongs in section: COSP Language: Fortran 95. This code is written to UMDP3 standards.

Parameters
[in]fluxPrecipitation flux is used
[in]x1These are the constants that define the PSD in the microphysics!> scheme (UMDP 26). X1,X3: n_ax = X1*exp(-X3*T[degC]) X2: n_bx X4: alpha_x
[in]aThese are the constants that define the mass-diameter relationship. M_x(D) = a_x*D^(b_x) A: a_x B: b_x
[in]cThese are the constants that define the terminal fall speed relationship. V_x(D) = c_x*D^(d_x)*exp(-h_x)*(rho_0/rho)^g_x C: c_x D: d_x G: g_x h_x is always 0 Abel and Shipway (2007) is not supported
[in]npointsDimensions
[in]tAir temperature [K]
[in]rhoAir density [kg/m^3]
[in]mrHydrometeor mixing ratio [kg/kg]
[out]reffEffective radius [m]

Compute intercept as function of T, if needed

Compute the parameter lambda^-1 of the PSD. stored in variable Reff
