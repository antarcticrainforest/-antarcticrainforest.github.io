First proceed forwards along the strip calculating horizon angles in the backwards direction

Set maximum horizon angle to the horizontal

Initially each point is its own horizon

Loop backwards from each point until we reach the horizon limit

Only consider points with higher orography

Horizon angle considers curvature of the earth

Record the new horizon angle and the point which forms it

If the horizon angle to this point is smaller (zenith to horizon) than the further horizon angle viewed from this point then we need not consider any further points.

If this point is its own horizon we can stop

If this point's own horizon is within our horizon limit then we can jump straight to the further horizon point

Otherwise we look to the next point along

Now proceed backwards along the strip calculating horizon angles in the forwards direction
