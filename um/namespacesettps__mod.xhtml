INTENT=OUT-!> ICODE : Return Code CMESSAGE : Reason for failure

The variables/arrays (all in comdeck COMACP) set up are : NACT : Number of AC Observation Types to be processed. LACT : List of the Obs Types in the order to be processed. N_GROUPS : Number of groups for processing in AC. GROUP_NO : Group in which each type is to be processed.

The order of processing is controlled by the array AC_ORDER.

The array AC_OBS_TYPES in the ACP namelist is used to control which observation types are to be processed.

Local variables

THIS_TYPE : Obs type in AC_ORDER J : Loop counter for obs types in LACT JOBT : Loop counter for obs types in AC_ORDER JTYPE : Loop counter for obs types in AC_ORDER THIS_GROUP : Indicator from AC_ORDER of grouping (current type) LAST_GROUP : Indicator from AC_ORDER of grouping (previous type)
