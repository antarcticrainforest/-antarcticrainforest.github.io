Subroutine Rcf_Locate - locates a field in a field array.

Fortran 90 replacement for locate1, using structures rather than a whole bunch of arrays. Will return the position of the field in the array if it exists otherwise will die gracefully. Optional argument zero_ok allows return of 0 in certain circumstances Method: Loop through the array until the correct field is found in section 0.
