Program MAKEBC : Top-level program to create boundary dataset.

Program MAKEBC : Top-level program to create boundary dataset from model analyses/dumps. Method : For each dump, boundary conditions are generated through GEN_INTF for the area specified in the INFTCNST namelist. This routine initialises various variables in TYPSIZE before it can be used in the lower routines. This file belongs in section: MakeBC
