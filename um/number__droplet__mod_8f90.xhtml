program __number_droplet_mod.f90__ ( )

Function to calculate cloud droplet number concentration. Function to calculate cloud droplet number concentration.

Purpose: Cloud droplet number concentration is calculated from aerosol concentration or else fixed values are assigned. Method: Sulphate aerosol mass concentration is converted to number concentration by assuming a log-normal size distribution. Sea-salt and/or biomass-burning aerosols may then be added if required. The total is then converted to cloud droplet concentration following the parametrization of Jones et al. (1994) and lower limits are imposed. Alternatively, fixed droplet values are assigned if the parametrization is not required.
