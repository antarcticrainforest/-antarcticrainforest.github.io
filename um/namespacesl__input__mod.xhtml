Detailed Description

Input control for SL dynamics. Input control for SL dynamics.

Module containing input switches/settings as used by the semi_lagrangian code and the check_run_sl routine for logic checking the selected settings. This file belongs in section: Dynamics Method: Switches are initialised to false and read in from the UMUI. The module may then be used directly where the switches are needed within the dynamics semi_lagrangian code. Language: FORTRAN 90 This code is written to UMDP3
