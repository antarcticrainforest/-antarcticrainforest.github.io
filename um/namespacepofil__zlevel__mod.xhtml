Detailed Description

Subroutine pofil_zlevel.

Subroutine pofil_zlevel Purpose: Filter/diffusion based on first order conservative diffusion operator. Applied as multiple EW 1-2-1 filter near poles. Applies z-level diffusion (truly horiztonal diffusion) Method: For any field. Pointers used to input grid. This version for u-at-the-poles new dynamics. ENDGame uses eg_pofil_vatp.!> This file belongs in section: Diffusion and Filtering Language: FORTRAN 90 This code is written to UMDP3 programming standards.
