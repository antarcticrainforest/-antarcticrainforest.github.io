◆ grborigecmwf

integer, parameter rcf_grib_lookups_mod::grborigecmwf = 98

The misc field is used to store header data not stored in the other lists.

=======================================================================

Parameters : The Originating centre no.s

These parameters correspond the the values of the Originating centres supplied by the GRIB record - Block 1 - Octet 1. They are used within rcf_grib_assign.F90 to select which column in table A (below) to use to find the variables STASH code from its given GRIB parameter code (Table 2 or Block 1 - Octet 5)
