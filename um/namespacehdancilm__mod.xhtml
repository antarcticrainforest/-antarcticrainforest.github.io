Subroutine HdAncilM - Count number of ANCILmaster reords.

Count the number of ANCILfields or ANCILfile records. Method: For an ANCILmaster file (including User files) read in all records and count number of records. The H4 label in the file indicates whether it is records for ancillary fields or files. The total number of records for fields and files are stored in max_ancRecs and max_ancFiles respectively.
