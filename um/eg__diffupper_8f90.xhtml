Subroutine diffupper Purpose: This routine used to evaluate diffusion at upper levels only. Calculates conservative horizontal diffusion of a field assuming that all model surfaces are at constant height. This means we can cancel the delta_eta terms and the r-terms can be absorbed into the diffusion coefficient. Method: Is described in ; This file belongs in section: Diffusion and Filtering Language: FORTRAN 77 + CRAY extensions This code is written to UMDP3 programming standards.
